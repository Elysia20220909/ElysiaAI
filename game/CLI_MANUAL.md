# Elysia Network Game CLI マニュアル

## ゲーム概要
- ノード（A, B, C）上をエージェント（P1, P2）が移動し、スコアを競うターン制ネットワークゲームです。
- 2人のユーザー（ユーザーID指定）が参加できます。
- 先にスコア10点に到達したユーザーが勝者となります。
- 同じノードに連続で移動するとペナルティ（スコア減点）が発生します。
- ターン制：同じユーザーが連続で行動することはできません。

## 操作方法（ターミナル版CLI）
1. サーバーを起動：
   ```
   bun ElysiaAI/game/server.ts
   ```
2. 別ターミナルでCLIクライアントを起動：
   ```
   bun ElysiaAI/game/cli-client.ts
   ```
3. プロンプトに従い、
   - あなたのユーザーID
   - 相手ユーザーID
   を入力します。
4. ゲームが開始されます。
5. 各ターンで、
   - 現在位置と移動可能ノードが表示されます。
   - 移動先ノード（A, B, Cのいずれか）を入力してください。
6. 勝者が決まるまでターンが進行します。
7. 勝者が決まるとゲーム終了です。

## ルール詳細
- ノード: A, B, C（それぞれ接続関係あり）
- エージェント: P1（あなた）、P2（相手）
- スコア: 移動ごとに+1、同じノード連続移動で-1
- 勝利条件: いずれかのユーザーがスコア10点に到達
- ターン制: 直前のユーザーと同じ場合はスキップ
- 履歴: 直近の行動履歴が毎ターン表示されます

## 注意事項
- サーバーとCLIクライアントは同じPCまたはネットワーク上で動作させてください。
- 複数ターミナルで同時にCLIクライアントを起動すれば、2人で対戦可能です。
- サーバー停止時は再度 `bun ElysiaAI/game/server.ts` で起動してください。
